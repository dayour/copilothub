# Copilot Hub Logo Set

Generated from the supplied Copilot Hub mark. The package includes vector SVGs, transparent PNGs, dark-background PNGs, favicon exports, horizontal and stacked lockups, wordmark-only variants, social-media assets, and a preview sheet.

## Recommended files

- Primary horizontal on light backgrounds: `svg/lockups/copilot-hub-horizontal-light.svg`
- Primary horizontal on dark backgrounds: `svg/lockups/copilot-hub-horizontal-dark.svg`
- Icon-only vector: `svg/icon/copilot-hub-icon-gradient.svg`
- App/profile tile: `png/icon/dark_background/copilot-hub-icon-dark-bg-1024.png`
- Favicon set: `png/favicon/`
- Open Graph/social preview: `png/social/copilot-hub-open-graph-1200x630.png`

## Usage notes

Use the SVG files for production wherever possible. Keep clear space around the logo equal to at least one quarter of the icon height. Use the light-background lockups on white or pale surfaces and the dark-background/white lockups on dark surfaces. Avoid stretching, recoloring the gradient arbitrarily, adding new shadows to the transparent SVG, or placing the gradient icon over low-contrast blue backgrounds.

The SVG wordmark references Inter with common system fallbacks. Font files are not included.
