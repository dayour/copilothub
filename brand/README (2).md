# Copilot Hub Full Icon Set

This package uses the SVG supplied in `Copilot_Hub_Logo_Set.zip` as the source mark, specifically:

`Copilot_Hub_Logo_Set/svg/icon/copilot-hub-icon-gradient.svg`

## Contents

```text
svg/gradient/        Gradient icons using the Copilot Hub slate-to-nitrous-blue palette
svg/mono-white/      White icons for dark UI surfaces
svg/mono-black/      Ink/black icons for light UI surfaces
svg/current-color/   CSS-colorable icons using currentColor
svg/copilot-hub-icon-sprite.svg  Inline SVG symbol sprite
preview/             Preview sheet
source/              Copy of the supplied source SVG and original manifest
```

- Night: `#0B1117`

## Usage

Use `svg/gradient/copilot-hub-hub.svg` as the primary product mark. For navigation or module icons, use the named module icons. Use `svg/current-color/` or `svg/copilot-hub-icon-sprite.svg` when the app theme should control the color through CSS.
