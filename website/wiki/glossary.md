# Glossary

| Term | Definition |
| --- | --- |
| **Action mode** | Chat tab mode where input is parsed as a tool command instead of conversation. |
| **Address bar** | Browser-tab control for entering URLs and search queries. |
| **CDP** | Chrome DevTools Protocol. Used by the sidecar to drive a real browser instance. |
| **CopilotBrowser** | The sidecar process exposing browser automation as MCP tools. |
| **Capability** | A Tauri 2 permission entry. |
| **Entra** | Microsoft Entra ID, the identity provider integrated via the SSO façade. |
| **Follow Me** | A browser sidecar mode in which the user's actions are mirrored into the automated browser context. |
| **Lockup** | The horizontal logo + wordmark combination used in the navbar and footer. |
| **MCP** | Model Context Protocol. |
| **Module icon** | A 24-token icon set covering each top-level surface (agents, automations, chat, integrations, etc.). |
| **NewTabPage** | The default landing surface for a new browser tab. |
| **Runbook** | A YAML workflow executed by `runbookExecutor.ts`. |
| **Sidecar** | Child process spawned and supervised by the Rust core. |
| **Tab type** | One of `browser`, `chat`, `terminal`, `vscode`, `runbook`. |
| **WebView2** | The Microsoft Edge–based rendering engine used by Tauri on Windows. |
| **Workbench** | The full CopilotHub application. |
